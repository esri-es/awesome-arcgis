Introduction

If you are used to work with jQuery you can load the jQuery library and use it instead of using Dojo to manage the DOM, Ajax, events, etc. Of course you can also use jQuery plugins, jQuery UI, jQuery Mobile, Bootstrap, Calcite Bootstrap, ... or any other technology built on top of jQuery.

Code

  • @Esri-es/arcgis-web-samples: code samples: ArcGIS API for JS + jQuery
  • @Esri/bootstrap-map-js: a light-weight JS/CSS extension for to combine Bootstrap with ArcGIS JS (3.x)
  • @Esri/dojo-bootstrap-map-js: how to use our API for JavaScript (3.x) with Bootstap via Dojo-bootstrap
  • @Esri/calcite-maps: a modern framework for designing map apps with Bootstrap
    • Starter app for the Esri Leaflet API + Bootstrap (jQuery) + Calcite
    • Starter app for the ArcGIS API for JS 4.x (jQuery)
  • jQuery mobile:
    • @Esri/jquery-mobile-map-js: dynamically resizes map when used in multi-view apps
    • @Esri/html5-geolocation-tool-js: use this fun jQuery Mobile web app to test HTML5 geolocation on different devices
