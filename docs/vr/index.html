Virtual Reality Developers

Export 3D scenarios built using City Engine, export them to Unity and build your application using any VR SDK to finish the application for your VR headset.

Introduction

This video can help you understand what can be done with ArcGIS and virtual.

To build you own applications using ArcGIS scenarios you first need to build the scenario using City Engine and then you will be able to export it and consume it using Unity.
